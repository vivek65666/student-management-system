# Student Management System

A responsive web-based Student Management System developed using HTML, CSS, and JavaScript. The application enables users to manage student records through Create, Read, Update, and Delete (CRUD) operations with data stored in Local Storage.

## Features

* Add Student Records
* Edit Student Information
* Delete Student Records
* Search Students by Name, ID, or Course
* Duplicate Student ID Validation
* Local Storage Data Persistence
* Responsive User Interface
* Client-side Form Validation
* Delete Confirmation Popup

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage
* Git
* GitHub

## How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/vivek65666/student-management-system.git
   ```

2. Open the project folder.

3. Double-click `index.html` or open it in any modern web browser.

4. Start adding, editing, searching, and deleting student records.

## Key Functionalities

### Add Student

Users can add new student records by entering:

* Student Name
* Student ID
* Course

### Edit Student

Existing student records can be modified using the Edit button.

### Delete Student

Records can be deleted with a confirmation popup to prevent accidental removal.

### Search Students

Users can search records dynamically using:

* Student Name
* Student ID
* Course

### Data Persistence

All student records are stored in the browser using Local Storage and remain available even after refreshing the page.
